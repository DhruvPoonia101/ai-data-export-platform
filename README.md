# AI-Powered Data Export & Analytics Platform

## Overview

A web-based platform that allows business users to connect databases, ask questions in natural language, generate SQL using AI, execute safe queries, and export results as CSV or Excel files.

## Tech Stack

### Frontend

* Next.js
* TypeScript
* ShadCN UI
* TanStack Query

### Backend

* FastAPI
* SQLAlchemy
* Alembic

### Database

* PostgreSQL

### AI

* OpenAI API

### Authentication

* JWT

### Infrastructure

* Docker
* Docker Compose
